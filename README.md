# Grocery List Manager Project
This is a Java project that manages a grocery list.
## How to run
Run the GroceryListManager.java file in Eclipse IDE.
## Features
- Add grocery item
- Remove grocery item
- View grocery list
- Exit

# Simple Banking System Project
This is a Java project that simulates basic banking operations through a text-based menu system.
## How to run
Run the BankApp.java file in Eclipse IDE.
## Features
- Check bank account balance
- Deposit money
- Withdraw money (with balance check)
- Exit

# Student Grade Tracker
This is a simple console-based Java application that allows users to manage and analyze student grades.
## How to run
Run the GradeApp.java file in Eclipse IDE.
## Features
- Add student grades
- View all entered grades
- Calculate average grade
- Find highest and lowest grade
- Exit
